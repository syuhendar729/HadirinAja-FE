# HadirinAja - Front End Mobile


## Step to install


1. Install dart, flutter, android studio, etc

Source: https://docs.flutter.dev/install/with-vs-code

2. Create .env

```
API_URL=https://urlbackend.you/api
```

3. Run:

```bash
git clone <url repo>
flutter pub get
<Command/Ctrl + Shift + P>
<Select Device>
flutter run
```

4. Shortcut:

```
<Shift + R> Restart
<R> refresh
Etc
```
